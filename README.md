# Pet Runner: Sticker Quest

A mobile-first canvas runner game about a house cat collecting paw scores, sticker cards, and health items while dodging household obstacles.

## Play

Open `index.html` in a browser, or serve the folder with any static web server.

The game is also installable as a PWA. On supported browsers, use the `로컬 설치` button on the result screen. On iOS Safari, use Share -> Add to Home Screen.

## GitHub Pages

This project is designed to run from the repository root on GitHub Pages. The game uses only static files:

- `index.html`
- `style.css`
- `game.js`
- `manifest.webmanifest`
- `sw.js`
- `assets/generated/`

No build step is required.
